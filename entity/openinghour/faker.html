Railken\Amethyst\Fakers\OpeningHourFaker

The faker can be used for testing or seeding.

Create a new entity using the faker

use Railken\Amethyst\Fakers\OpeningHourFaker;

$result = $manager->create(OpeningHourFaker::make()->parameters());

Extend the class

Create the new faker in app/Fakers/OpeningHourFaker

namespace App\Fakers;

use Railken\Amethyst\Fakers\OpeningHourFaker as Faker;

    // ...
}

Update the file configs/amethyst.opening-hour with the new class

return [

            'faker' => App\Fakers\OpeningHourFaker::class,
        ],
    ]
];
